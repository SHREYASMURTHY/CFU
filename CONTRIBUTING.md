# Contributing to Bacterial Colony Counter

Thank you for your interest in contributing to this project! We welcome contributions from the community to help make this platform better.

## How to Contribute

### 1. Reporting Bugs

- Check the [Issues](https://github.com/RohanRajesh55/Bacterial-Colony-Counter/issues) to see if the bug has already been reported.
- If not, create a new issue with a clear description, steps to reproduce, and any relevant logs or screenshots.

### 2. Suggesting Enhancements

- Open an issue and label it as an "enhancement."
- Describe the feature you'd like to see and why it would be useful.

### 3. Pull Requests

- Fork the repository.
- Create a new branch for your feature or fix.
- Ensure your code follows the existing style and is well-documented.
- Submit a pull request with a detailed explanation of your changes.

## Development Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/RohanRajesh55/Bacterial-Colony-Counter.git
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    cd web/frontend && npm install
    cd ../backend && npm install
    ```

## Code of Conduct

Please be respectful and constructive in all interactions within this project.
